## **express란?**

: Node.js의 핵심 모듈인 http와 Connect 컴포넌트(미들웨어라고 부른다)를 기반으로 하는 웹 프레임워크

## **express의 특징**

- 최소화
  - express는 사용자에게 최소한의 프레임 워크 제공
- 유연함
  - client HTTP 요청을 받고, 응답 반환
- 속도
  - 트래픽이 많은 웹 사이트에서 최고의 성능 발휘
- 크기와 다양성
  - 개발 시 원하는 구성요소를 쉽게 추가 가능하도록 유연한 미들웨어 시스템을 만듦
